- **개요**: 온프레미스 환경의 '법카 플렉스' 서비스를 AWS 클라우드로 마이그레이션하고, MSA 구조에 기반한 고가용성 및 확장성 있는 인프라를 구축했습니다. 또한, 챗봇 기능 추가 및 운영 효율화를 위한 통합 모니터링 대시보드를 구현했습니다.
