# 📘 phpmyadmin

dns서버에 이어서 phpmyadmin을 실습

아파치 기본 폴더에 phpadmin을 다운로드
```
wget https://files.phpmyadmin.net/phpMyAdmin/4.0.10.20/phpMyAdmin-4.0.10.20-all-languages.zip

```
그리고 파일압축을 푼다.
```
unzip phpMyAdmin-4.0.10.20-all-languages.zip
```
