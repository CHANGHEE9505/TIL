## 첫 번째 플레이북 작성하기
### 플레이북 환경 설정 
- 'Ansible'(root/my-ansible/ansible.cfg)
 
```
[root@localhost my-ansible]# vi ansible.cfg
[root@localhost my-ansible]#
[root@localhost my-ansible]# cat ansible.cfg
[defaults]
inventory = ./inventory
remote_user = user
ask_pass = false

[privilege_escalation]
become = true
become_methode = sudo
become_user = root
become_ask_pass = false
```
